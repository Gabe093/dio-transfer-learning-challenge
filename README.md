# dio-transfer-learning-challenge
Transfer-learning project using VGG16 for classifying eagles and chikens - DIO Challenge

About this Project:
This project is part of the machine learning course offered by the DIO platform. The main goal of this challenge was to create a Python script and apply the transfer learning method to a deep learning network. After training, the model was able to classify images of eagles and chickens from a dataset.

How to use:
From a database organized into training, validation, and test folders, replace the file path in the variable 'root_dir = './eagles_chickens_organized/'' so that the model initially trains a network from scratch and subsequently uses transfer learning. Finally, compare the performance between the two models using the graphs.
